# Breast Cancer Surgery Workshop Website

## Overview
A beautiful two-page website for a breast cancer surgery workshop designed to empower healthcare professionals through advanced surgical training.

## Pages
1. **Home Page (index.html)** - Features animated navbar, image slideshow, expert profiles, and inspiring footer
2. **Sign-In Page (signin.html)** - Professional registration form for workshop participants

## Design Features
- Calm purple and pink color scheme representing breast cancer awareness
- Smooth animations and transitions throughout
- Responsive design for all devices
- Elegant typography and spacing
- Professional imagery from Unsplash

## Color Palette
- Primary: #d946b2 (Pink/Magenta)
- Secondary: #fdf4ff (Light Purple)
- Accent: #a21caf (Deep Purple)

## Key Components
- Navbar with scroll effects
- Auto-playing slideshow with fade transitions
- Animated expert profile cards
- Multi-field registration form with optional image upload
- Inspirational footer with encouraging message

## Technology Stack
- React 18
- Tailwind CSS
- Lucide Icons
